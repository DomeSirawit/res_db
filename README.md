# res_db

create restaurant database
this database example consist of 1 fact table and 4 dimension table (star schema) 

star schema is the simplest and most use in the data warehouse among all type of schema, the schema model resembles to the star shape having a fact table at its center and the dimension tables at its peripheral representing the starâ€™s points

Dimension tables : describe business entities

Fact table : contain events, facts about a business process
